# renju

## 五子棋

1. 棋盘大小为 15*15
2. 主持黑，黑棋先手，黑先白后
   > 如果是黑方的第一回合，则 request 为 {"x": -1, "y": -1}）
3. 在黑棋第三手后（棋盘上有两黑一白三个棋子），白棋可以选择是否换手
   > 输出 response 为 {"x": -1, "y": -1}，代表换手。
   > 如果换手，则黑棋变白白棋变黑，换手后下一手由黑棋下。
4. 落子位置 "x" 横坐标, "y" 纵坐标

详见 [botzone wiki](https://wiki.botzone.org.cn/index.php?title=Renju)
